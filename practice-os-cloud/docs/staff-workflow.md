# Staff Workflow

## Daily

1. Open `/my-work`.
2. Complete overdue and due-today tasks first.
3. For each task:
   - update status
   - enter time
   - add remarks
   - upload work files
   - mark checklist points
4. End of day:
   - ensure all worked tasks have time entries
   - mark data-awaited tasks clearly
   - send prepared items for review

## Weekly

Manager/reviewer checks:

- overdue tasks
- staff with high open task count
- review pending tasks
- clients delaying data
- high time clients

## Monthly

Owner checks:

- completed work
- pending carried forward
- billing and outstanding
- low-profit clients
- staff utilization
- fee revision list

## Status Discipline

Use statuses consistently:

- `DATA_AWAITED`: waiting for client data
- `IN_PROGRESS`: staff is working
- `PREPARED`: ready for reviewer
- `UNDER_REVIEW`: reviewer checking
- `CORRECTION_REQUIRED`: staff must fix
- `READY_FOR_SUBMISSION`: client/reviewer approved
- `FILED_SUBMITTED`: statutory filing/submission done
- `COMPLETED`: all work and acknowledgement complete
